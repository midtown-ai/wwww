## Content Tabs

### simple

=== "Tab A"
    In a different tab set.

=== "Tab B"
    More content.

```
=== "Tab A"
    In a different tab set.

=== "Tab B"
    More content.

```

### Linked tabs

=== "First"

    This is the content of my first tab!

=== "Second"

    Content of second tab

===! "First"

    Tab-names must match

    New tab set is started with `===!`

=== "Second"

    Tab-names is same as other tab above
